# 🌾 AgriNex Frontend - Animated Agri-Tech Solution

Welcome to the **AgriNex Frontend**, a premium, high-performance, and **fully animated** web application designed to empower farmers with AI-driven agricultural insights.

Built with **React 19** and **Vite**, this frontend features a "Fresh Organic Agri-Tech" aesthetic with smooth transitions, interactive components, and a modern dark/light forest theme.

---

## ✨ Key Features (Animated & Interactive)

### 🚀 High-End Animations
- **Framer Motion**: Every page features scroll-triggered entrance animations, hover effects, and fluid layout transitions.
- **Lottie React**: Integrated Lottie animations for a lively and engaging user experience (e.g., hero section, loading states).
- **Interactive Cards**: Feature cards with subtle hover scaling and shadow depth.

### 🛠️ Core Modules
- **🏠 Animated Home**: A stunning landing page with a hero section powered by Lottie and Framer Motion.
- **🌱 Crop Prediction**: Predict the best crops to grow based on soil parameters (N, P, K, pH) and location-specific weather.
- **🧪 Fertilizer Recommendation**: Get tailored fertilizer advice based on soil nutrient levels and target crop.
- **📸 Disease Detection**: Upload plant leaf images for instant AI-powered disease diagnosis and treatment advice.
- **🌤️ Nutrient & Weather Insights**: Visualize soil nutrient needs alongside real-time 5-day weather forecasts.

---

## 🛠️ Tech Stack

- **Framework**: [React 19](https://reactjs.org/) (Vite)
- **Styling**: [Bootstrap 5](https://getbootstrap.com/) + Custom Vanilla CSS
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [Lottie React](https://github.com/gamote/lottie-react)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- Backend server running (Flask)

### Installation
1. Clone the repository and navigate to the `Frontend` folder:
   ```bash
   cd Frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.
